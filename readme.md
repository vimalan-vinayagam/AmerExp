# Selenium Test Automation Framework

## 📌 Overview
This framework is built using **Java + Selenium + TestNG + Cucumber** for automated UI testing. It supports **parallel execution, reporting, and data-driven testing**.

## 🔧 Technologies Used
- **Selenium WebDriver**
- **Java 11+**
- **TestNG**
- **Cucumber (BDD)**
- **Maven**
- **Extent Reports**
- **WebDriverManager**
